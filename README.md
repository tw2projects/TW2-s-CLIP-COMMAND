# TW2-s-CLIP-COMMAND
TW2ClipTool ist C# Inline Code für Streamer.bot (CPHInline), der zwei Dinge abdeckt. Erstens führt es eine Twitch Authentifizierung per Device Code Flow durch und speichert die erhaltenen Tokens als Streamer.bot Globals. Zweitens erstellt es per Command einen Twitch Clip über den Helix Endpoint /helix/clips
